# Project Description
NSR Parfum Store is a modern e-commerce website created using Shopify.  
The website is designed for selling premium perfumes and fragrance products online. Customers can browse perfume collections, check product prices, view product images, and contact the store directly through the contact form.

The website was developed with a clean and responsive design to provide a smooth shopping experience on both desktop and mobile devices.

---

# Platform Used
- **Shopify**
- **GitHub**

---

# Features Implemented
The project includes the following features:

- Homepage with welcome banner
- Product collection section
- Product page with perfume images and prices
- Contact page with form
- Privacy Policy page
- Responsive mobile-friendly design
- Modern and simple UI/UX
- Add-to-cart interaction
- Navigation menu for easy browsing
